This project is part of the KAIM Week 0 challenge. It focuses on analyzing climate data for East African countries.

 Project Structure
- notebooks/ : Jupyter notebooks for analysis
- cleaned_data/ : Processed datasets
- plots/ : Visualizations
- data/ : Raw datasets
- requirements.txt : Python dependencies

 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

#Tasks Completed
- Environment setup using Git and virtual environment
- Data profiling and cleaning
- Initial exploratory data analysis (EDA)
