# Climate Data Analysis Project

## Overview
This project analyzes climate data (temperature, rainfall, humidity) across multiple African countries to identify trends, variability, and climate risks.

## Project Structure

data/
 ├── raw/          # Original datasets
 └── processed/    # Cleaned datasets generated by code

notebooks/
 └── analysis.ipynb   # Main analysis notebook
plots/
 └── (generated visualizations)
##  Methodology

- Data cleaning:
  - Missing values handled using forward fill
  - Invalid values (-999) replaced with NaN
  - Duplicates removed

- Feature engineering:
  - Created Date, Month, Year
  - Calculated temperature range

- Analysis:
  - Time series trends
  - Monthly aggregation
  - Correlation analysis
  - ANOVA test for country comparison
## How to Run

1. Place raw data in data/raw/
2. Run the notebook:
   notebooks/climate_analysis.ipynb
3. Outputs will be generated automatically:
   - Cleaned data → data/processed/
   - Plots → plots/

## Outputs

The following outputs are automatically generated when running the analysis notebook:

- Cleaned Data  
  Saved in data/processed/  
  Generated using the data cleaning functions in the notebook.
- Plots / Visualizations  
  Saved in plots/  
  Created using matplotlib and seaborn during the analysis phase.
- Statistical Results  
  Includes summary statistics and ANOVA test results printed in the notebook.

### How outputs are generated

1. Raw data is loaded from data/raw/
2. Data is cleaned (missing values handled, duplicates removed)
3. Cleaned data is saved to data/processed/
4. Visualizations are created and saved to plots/
